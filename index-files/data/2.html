الگوی هرمی

الگوی مثلّثی جمع اعداد صحیح است؛ بنابرین فاصله‌ی جمله‌ی n و n+1 برابر با n+1 است. همینطور فاصله‌ی جمله‌ی n و n+1 الگوی هرمی برابر با 2/(n+1)*(n+2) است. برای اینکه فاصله‌ی الگوی هرمی 2/(n+1)*(n+2) شود؛ باید هم جمله‌ی n و هم جمله‌ی n+1 مضرب 2/(n+1)*(n+2) باشند. در نتیجه جمله‌ی n شامل n و جمله‌ی n+1 شامل n+3 نیز هست. تا اینجا جمله‌ی n+1 برابر است با 2/(n+1)*(n+2)*(n+3) و جمله‌ی n برابر است با 2/(n)*(n+1)*(n+2) حال اینها را از هم کم می‌کنیم که جواب 3/2*(n+1)*(n+2) می‌شود. برای همین اگر فرمول جمله‌ی n را بر سه تقسیم کنیم حاصل 2/(n+1)*(n+2) می‌شود. پس فرمول جمله‌ی n الگوی هرمی برابر است با:

n*(n+1)*(n+2)/6
